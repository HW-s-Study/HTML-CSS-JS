### 하이퍼링크 삽입하기
- <a href=“링크할 주소”>텍스트or이미지</a> : 텍스트는 단순 입력, 이미지는 <img> 태그 사용 <br />
ㄴ <a href=""></a> : 텍스트 입력 <br />
ㄴ <a href=""><img src="" alt=""></a> : 이미지와 설명 입력 <br />

!! target="_blank"를 이용해 새 창에서 열도록 할 수 있음 <br />
-> target 속성은 이 링크가 어디에서 열릴지 선택할 수 있게 해줌 <br />



### 입력 양식(form) 사용하기
- <form> - </form> : 특정 항목에 사용자가 뭔가를 입력할 수 있게 만든 것, 폼을 만드는 가장 기본적인 태그 <br />
- <input type=""> : form에서 사용자가 입력한 정보를 받기 위한 필드 <br />
ㄴ method : 사용자가 입력한 내용을 서버로 어떻게 넘겨줄 것인지 지정, get과 post 방식 <br />
!! get의 경우 사용자가 입력한 내용이 주소에 나타나는 반면 post의 경우 나타나지 않음 <br />

ㄴ action : <form> 태그 안의 내용을 전송할 서버의 주소, 현재는 잘 사용하지 않고 대부분을 JS에 위탁해 한번에 처리 <br />
ㄴ name : 같은 속성의 태그를 구분하기 위한 이름, JS로 폼을 제어할 때 사용할 폼의 이름을 지정 <br />
ㄴ target : action 속성에서 지정한 스크립트 파일을 다른 위치에서 열도록 함(위의 target과 동일) <br />
ㄴ autocomplete : 브라우저가 이전에 입력했던 값을 기억해서 자동으로 표시, 자동완성(기본 속성값 on) <br />


- <fieldset> : 하나의 폼 안에서 여러 구역을 나누어 표시할 때, 선으로 나누어짐(form 태그 바로 아래에 사용) <br />
- <legend> : fieldset으로 묶은 그룹에 제목을 붙임, fieldset 선 부분에 제목으로 표시됨(fieldset 태그 안에 사용) <br />


- <label> : 폼 요소와 레이블 텍스트가 연결되었다는 것을 웹브라우저에게 알림(태그와 텍스트에 대한 설명) <br />
ㄴ <label for="">, <input id=""> 형태로 같은 id를 주어 연결하거나 label 태그로 input 태그를 감싸 연결 <br />


- <textarea> : 여러 줄을 입력하는 텍스트 영역 <br />
ㄴ cols="" : 가로 너비를 문자 단위로 지정 <br />
ㄴ rows="" : 세로 너비를 문자 단위로 지정, 지정한 숫자보다 줄 개수가 많아지면 스크롤 바가 생김 <br />

- <button> : 버튼 입력을 위한 버튼의 형태를 표시(기능은 없는 빈 껍데기) <br />



#### select & option
- <select><option> : Dropdown List(콤보 박스), 옵션에 있는 여러 종류들을 선택하기 쉽도록 드롭다운 메뉴로 표시 <br />
ㄴ <select name=""> : name 속성을 이용해 선택된 값(option)을 서버로 전송 <br />
ㄴ <select multiple> : controls를 이용해 다중 선택이 가능하도록 하는 속성 <br />
ㄴ <select size=""> : 화면에 표시할 옵션의 개수를 지정 <br />

ㄴ <option value=""> : name의 정보는 컴퓨터가 모르기에 서버가 정보를 이해할 수 있도록 값을 전송 <br />
ㄴ submit : 폼에 작성된 내용을 action 주소의 서버로 전송 <br />
ㄴ reset : 폼에 작성된 내용을 초기화 <br />
```
<select name="">
  <option>값</option>
</select>
```
!! 공통적으로 value="" 값을 주어 버튼 내부의 글자를 바꿀 수 있음
!! selected를 사용해 드롭다운 메뉴에서 기본적으로 선택해서 보여줄 항목을 지정할 수 있음


#### datalist & option
- <datalist><option> : input에 자동완성용 선택 목록을 붙여주는 태그, autocomplete와 달리 개발자가 만든 리스트 기반으로 자동완성<br />
```
<input type="text" list="데이터목록">
  <datalist id="데이터 목록">
  <option value="서버로 넘길 값">선택옵션1</option>
</datalist>
```

-> 이후 추가적인 태그의 속성들은 [00]학습자료 폴더의 HTML 학습지 마지막 페이지를 참고
