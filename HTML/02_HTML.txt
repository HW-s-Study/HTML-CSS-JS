### 표 만들기
- <table> - </table> : table(표), 태그 내부에 있는 요소들이 하나의 표를 이루게 해주는 태그 <br />
- <th> - </th> : table head(표 제목), 표를 이루고 있는 데이터 요소 중 제목에 해당하는 요소 <br />
- <td> - </td> : table data(표 데이터), 표를 이루고 있는 각각의 데이터 요소 <br />
- <tr> - </tr> : table row(표 행), 각각의 요소들의 한 행 <br />

!! <table> 태그 바로 아래 위치하는 대분류 태그, 작성하지 않아도 브라우저가 인식해 작성해주기에 큰 지장은 없음
- <thead> - </thead> : 표의 머리를 뜻하는 태그 <br />
- <tbody> - </tbody> : 표의 몸통을 뜻하는 태그로 대부분의 데이터가 이 태그에 속함 <br />
- <tfoot> - </tfoot> : 표의 꼬리를 뜻하는 태그로 상위 태그들과 위치가 섞여도 무조건 최하단에 위치 <br />


#### table 태그 스타일링
<table border=1>와 같은 형태로 간단하게 표에 테두리를 줄 수 있지만, 추후 사용할 css라는 스타일링 언어를 이용해 스타일링한다 <br />
과거에는 표를 사용해 레이아웃을 구성하기도 하였으나 현재는 권장하는 방법이 아니기에 사용하지 않는다 (but. 과거에 사용한 방식으로 단순히 표 제작만이 가능한건 아님) <br />
