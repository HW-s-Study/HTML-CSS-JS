ex. 'css text size property'(글자크기 속성) <br />

### 글꼴 관련 스타일
- font-size : 글자 크기 지정 <br />
절대적 = <br />
ㄴ px : 컴퓨터에 정해진 px 단위를 사용해 폰트 사이즈를 지정 <br />
가변적 = <br />
ㄴ rem : html 기준(기본 16px) n배의 사이즈로 폰트 사이즈를 지정 <br />
ㄴ em : 부모 요소를 기준으로 n배의 사이즈로 폰트 사이즈를 지정 <br />
!! em은 중첩될수록 크기가 눈덩이처럼 불어나므로 유의 (부모: 1.5 × 기본 16px = 24px, 자식: 1.5 × 24px = 36px) <br />
!! 현재는 사용자가 글자 크기를 조정할 수 있도록 하는 권리를 주기 위해 px를 써야만 하는 이유가 있지 않다면 가변적인 단위인 rem, em을 사용 <br />

- font-family : 글꼴 지정의 역할, 여러개 지정해 폰트가 없을 경우 차선책 제공 가능(폰트 이름에 띄어쓰기가 있는 경우 따옴표로 묶기) <br />
ㄴ monospace : 고정폭 글꼴 <br />
ㄴ selif : 장식 있는 글꼴 <br />
ㄴ sans-selif : 장식 없는 글꼴 <br />
ex. ```font-family: arial, verdana, '맑은 고딕';```

- font-weight : 글자 굵기 지정(normal(400), bold(700), bolder, lighter, 100~900) <br />

- font-style : 이탤릭체로 글자 표시 (normal, italic, oblique) <br />
!! oblique는 강제 이탤릭으로 이탤릭체가 없는 글꼴이어도 강제로 기울이는 효과 <br />


### 텍스트 관련 스타일
- color : 글자색 지정 <br />
ㄴ 색상 이름 : 유명한 색상들을 영문 텍스트로 지정 <br />
ex. ```color: red;```
ㄴ hsl, hsla(색상, 채도, 명도, 불투명도) : 0 ~ 360의 각도(색상환)으로 색상을 표현(채도와 명도는 %로 표현) <br />
ㄴ rgb, rgba(red, green, blue) : 색의 삼원색을 기준으로 각각 조절해 색상을 표현(a는 Alpha로 투명도를 의미) <br />
ㄴ 16진수(#RRGGBB) : rgd를 16진수로 표현 <br />

- text-align : 텍스트 정렬 <br />
ㄴ start, end : 현재 텍스트 줄의 시작 or 끝 위치에 맞추어 문단을 정렬(기본은 start) <br />
ㄴ left : 왼쪽 정렬 <br />
ㄴ right : 오른쪽 정렬 <br />
ㄴ center : 가운데 정렬 <br />
ㄴ justify : 양쪽 여백에 맞추어 정렬 <br />
ㄴ match-parent : 부모 요소를 따라 문단 정렬 <br />
